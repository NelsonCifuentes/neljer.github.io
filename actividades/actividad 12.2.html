Background Color - Color de Fondo

Si no se especifica un colo de fondo, el fondo es transparente.

CSS trata cada elemento HTML como un cuadro y la propiedad background-color establece el color del fondo para el cuadro. Se peude especificar un color de Fondo de las mias tres formas en que peude especificar colores de primer plano: Valores RGB, Codigo Hexadecimal y Nombre de Colores. Tambien hemos utilizado la propiedad padding para separa el texto de los bordes de cuadros. Esto hace que sea mas facil de leer.
