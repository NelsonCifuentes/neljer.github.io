Adding HTML5 Video

Un video de un cachorro jugando en la nieve

En HTML5 no es necesario proporcionar valores para todos los atributos, Como los controles, reproducción automática y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido/apagado. Si el atributo está presente, activa esa opción. Si se omite el atributo, la opción esta desactivada. Si el navegador no admite el elemento <video> o el formato video utilizando, se mostrara lo que este entre la <video> de apertura y las etiquetas de < /video> de cierre

preload: este atributo le indica al navegador que hacer cuando se carga la pagina. Puede tener uno de tres valores:
none: el navegador debe cargar el video hasta que el usuario presiona Play
auto: el navegador debe descargar el video cando se carga la pagina.
metada: el navegador debe simplemente recopilar informacion como el tamaño, el primer fotograma, la lista de pistas y la duracion.
